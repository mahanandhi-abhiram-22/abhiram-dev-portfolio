# Abhiram Portfolio

Premium MERN Stack Developer Portfolio built with React, Vite, Tailwind CSS, and Framer Motion.

## Features
- Glassmorphism UI Design
- Smooth animations with Framer Motion
- Fully responsive design
- Live project showcases
- Contact form integration
- Modern developer portfolio

## Tech Stack
- React 18+
- Vite (build tool)
- Tailwind CSS
- Framer Motion
- React Icons

## Quick Start

1. Clone the repository
```bash
git clone https://github.com/mahanandhi-abhiram-22/abhiram-dev-portfolio.git
cd abhiram-portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```

## Sections
- **Hero** - Eye-catching introduction
- **About** - Personal background and skills
- **Skills** - Technical proficiency showcase
- **Experience** - Professional journey
